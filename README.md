  - 1주일에 총 7문제(월~일)
  - 타임리밋 및 코드리뷰 : 일요일
- 10/19~11월 중순 : easy
- 11월 중순 ~ : medium
- 난이도는 진행하면서 조절
- 리포지토리 디렉토리 구조 예시
